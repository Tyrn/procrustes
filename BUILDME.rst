Procrustes SmArT
****************

Audio album builder; copy and edit on the fly.

Development
===========

- `Format Rust code <https://github.com/rust-lang/rustfmt>`__

Build
-----

::

    $ cargo build [--release]

Format
------

::

    $ cargo fmt

Test
----

::

    $ cargo test

Install
-------

Install to ``~/.cargo/bin``:

::

    $ cargo install --path . [--debug]
